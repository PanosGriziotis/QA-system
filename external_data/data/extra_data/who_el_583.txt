Η συζήτηση αυτή θα μπορούσε επίσης να γίνει με τον υπεύθυνο για τους ανθρώπινους πόρους στην εργασία σας.Υπάρχουν πολλά πράγματα που μπορείτε να κάνετε για να αισθάνεστε πιο άνετα στη νέα σας θέση εργασίας, ενώ εργάζεστε εξ αποστάσεως. Ζητήστε από τον προϊστάμενό σας να έχει πρόσβαση σε μια εκπαίδευση σχετικά με την εργασία από απόσταση και τις δεξιότητες προσωπικής αποτελεσματικότητας. Η αξιοποίηση του χρόνου σας για την πρόσβαση σε ευκαιρίες μάθησης και κατάρτισης μπορεί όχι μόνο να σας βοηθήσει να αναπτύξετε δεξιότητες για τον τρέχοντα ρόλο σας αλλά και να σας προετοιμάσει για μελλοντικούς ρόλους που μπορεί να επιθυμείτε. Συμφωνήστε με τον προϊστάμενό σας να πραγματοποιείτε τακτικές κλήσεις ελέγχου, όπου μπορείτε να ενημερώνετε τον προϊστάμενό σας για τα επιτεύγματά σας με μια ενημερωτική περίληψη. Εάν έχετε τακτικές ομαδικές συναντήσεις, χρησιμοποιήστε τις ως ευκαιρία για να επισημάνετε τις βασικές σας επιτυχίες και να γνωρίσετε καλύτερα τους συναδέλφους σας.
