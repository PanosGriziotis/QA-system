Μπορεί να είναι καταλληλότερη για να καθοδηγήσει την κατανομή των σπάνιων πόρων που παρέχουν ουσιαστικά διαφορετικά οφέλη σε διαφορετικά άτομα, για παράδειγμα, αναπνευστήρες σε εκείνους που αναμένεται να αποκομίσουν το μεγαλύτερο όφελος τους χειρότερουςΑυτή η αρχή μπορεί να χρησιμοποιηθεί για να δικαιολογήσει την κατανομή των πόρων σε εκείνους που έχουν τη μεγαλύτερη ιατρική ανάγκη ή εκείνους που διατρέχουν τον μεγαλύτερο κίνδυνο είναι καταλληλότερη για να καθοδηγήσει την κατανομή των πόρων που έχουν σχεδιαστεί ή προορίζονται για την προστασία εκείνων που διατρέχουν κίνδυνο, για παράδειγμα, ΜΑΠ για τους εργαζόμενους στον τομέα της υγειονομικής περίθαλψης, εμβόλια για εκείνους που διατρέχουν τον μεγαλύτερο κίνδυνο μόλυνσης και σοβαρής ασθένειας ή εκείνους που έχουν τη μεγαλύτερη ανάγκη, όπως στην περίπτωση της παροχής φαρμάκων που βρίσκονται σε ανεπάρκεια σε όσους τα χρειάζονται πιο επειγόντως όσοι έχουν αναλάβει να βοηθήσουν άλλουςΗ αρχή αυτή μπορεί να χρησιμοποιηθεί για να δικαιολογήσει την κατανομή πόρων σε όσους έχουν ορισμένες δεξιότητες ή ταλέντα που μπορούν να σώσουν πολλούς άλλους ανθρώπους, ή επειδή τους οφείλεται κάτι λόγω της συμμετοχής τους στην παροχή βοήθειας σε άλλους να είναι η καταλληλότερη για να καθοδηγήσει την κατανομή των πόρων σε εργαζόμενους στον τομέα της υγειονομικής περίθαλψης, σε υπηρεσίες πρώτης ανταπόκρισης, κ.λπ. οι αρχές μπορεί να είναι σχετικές ή να δικαιολογούνται σε διαφορετικά στάδια έλλειψης πόρων (από τη μικρότερη έλλειψη στη μεγαλύτερη έλλειψη).
